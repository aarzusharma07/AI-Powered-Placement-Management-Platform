## 🏗️ Technology Stack (Modern SaaS)

### Frontend Tier
- **Framework:** React.js + Next.js 14+ (SSR, optimized bundling)
- **Styling:** Tailwind CSS (utility-first, responsive)
- **HTTP Client:** Axios (REST API calls)

### Backend Tier
- **Runtime:** Node.js (event-driven, non-blocking I/O)
- **Framework:** Express.js (routing, middleware)
- **Auth:** JWT + Passport.js (stateless tokens, RBAC)

### Database Tier
- **Primary:** MongoDB Atlas (document store, flexible schema)
- **Cache:** Redis (in-memory for performance, queue management)
- **File Storage:** AWS S3 (resume documents, compliance)

### AI Services Tier
- **Language:** Python + FastAPI (industry standard for NLP)
- **Resume Parsing:** spaCy + PyMuPDF (Named Entity Recognition)
- **Vector Matching:** SentenceTransformers + FAISS (semantic similarity)
- **LLM Integration:** OpenAI API (question generation, analysis)

### Infrastructure
- **Hosting:** AWS ECS/Fargate (containerized, auto-scaling)
- **CDN:** CloudFront (global asset distribution)
- **Database:** MongoDB Atlas (managed, replicated)
- **Backup:** Automated daily snapshots (30-day retention)

## 💡 5 Key Innovations

1. **Semantic Candidate Matching**
   - Converts job descriptions & resumes to vector embeddings
   - Calculates similarity scores (0-100%) based on semantic meaning, not keywords
   - Result: 40%+ application conversion rate (vs. 20% baseline)

2. **Automated Resume Parsing**
   - Uses NER (Named Entity Recognition) + spaCy
   - Extracts: Name, Education, Skills, Projects, Experience
   - Result: 90%+ accuracy, 5-second processing time

3. **Smart Interview Scheduling**
   - Cross-references candidate + recruiter calendars
   - Auto-suggests conflict-free time slots
   - Generates calendar invites (Google Calendar, Outlook)
   - Result: 4-6 hours saved per drive

4. **Real-Time Compliance Reporting**
   - NIRF/NAAC audit reports generated in 1 click
   - Structured data export (PDF + Excel)
   - Eliminates 3-week manual compilation process
   - Result: 1 hour vs. 3-4 weeks

5. **Predictive Job Recommendations**
   - Analyzes student profile + past hiring trends
   - Ranks jobs by likelihood of offer
   - Pushes top 5 matches to student dashboard
   - Result: Higher engagement + faster placement
